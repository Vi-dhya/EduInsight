## Email Format Requirements

### Faculty Email Format
- Format: `faculty[name]@college.edu`
- Example: `facultyrajesh@college.edu`
- Name must contain only letters (a-z, A-Z)
- No numbers or special characters in the name part

### Student Email Format
- Format: `student[8-digits]@college.edu`
- Example: `student23102060@college.edu`
- Must have exactly 8 digits

## Password Requirements
- Minimum 8 characters
- Must include uppercase letter
- Must include lowercase letter
- Must include number
- Must include special character

## How to Login

1. Go to the login page
2. Enter faculty email (e.g., `facultyrajesh@college.edu`)
3. Enter password (e.g., `Faculty@123`)
4. Click Login
5. You will be redirected to the Faculty Dashboard

## Faculty Dashboard Features

- View all students by year and department
- View student details with ABC ID and Income Certificate status
- Manage certifications and internships
